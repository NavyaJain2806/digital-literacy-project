## Task 3 – Coding & Collaboration Platforms

### Part A – Coding Practice (HackerRank)
I created an account on HackerRank and completed the 'Solve Me First'
challenge in the Algorithm domain. The problem required writing a
function that adds two integers. Completing it helped me get familiar
with the HackerRank interface and submission system. A screenshot of
the completed challenge is included in this folder.

### Part B – Google Forms Quiz
I built a five-question 'Digital Literacy Awareness Quiz' using Google
Forms for my batchmates. The questions are:

1. What does the term 'digital literacy' mean? (Multiple choice)
2. Which of the following is a version control platform used by
   developers? (Multiple choice)
3. Name one best practice for staying safe online. (Short answer)
4. What is phishing? (Multiple choice)
5. Which Google tool would you use to collaboratively edit a document
   with your classmates? (Multiple choice)

Responses are automatically collected in a linked Google Sheet.
The form link is also available in the main README.
